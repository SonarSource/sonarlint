This version of SonarLint brings a new metric (Cognitive complexity) and fixes the calculation of the Cyclomatic Complexity. Here are the changes since version 2.10.0:

  • Cognitive Complexity - SONARCS-661
  • Fix Cyclomatic Complexity - SONARCS-630
  • Fix False-Positive on rule S2674 - SONARCS-674
  • Abstract classes should not have public constructors - SONARCS-690
